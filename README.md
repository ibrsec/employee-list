<a name="quick-setup"></a>
## 🛫 Quick Setup

```sh
# clone the project
git clone https://github.com/ibrsec/employee-list.git

# enter the project directory
cd employee-list

# install dependency
npm install || yarn install

# develop
npm run dev || yarn start
```

## 📂 Directory structure 

```diff

  |     |

  |     |     |    
  |     |     |---header (folder) 
  |     |     |     └---Header.jsx 
  |     |     |
```
